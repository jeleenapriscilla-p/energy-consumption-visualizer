About This Project

The Energy Consumption Visualizer is a front-end web application designed to calculate and display the energy usage of common household appliances. Users can add devices, specify wattage and usage hours, and instantly view the resulting kWh and cost estimations. This project demonstrates HTML5 structure, CSS3 UI styling, and JavaScript for dynamic behavior.
